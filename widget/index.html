This plugin and this setup is responsible for showing clear steps how to reproduce core bugs. The goal of this plugin is for core team to have enough details and exact code steps on how to reproduce the issue. Developer who reported core bug is now responsible to use this template, fill all placeholders "{ }" carefully with full details and create minimal but yet enough explainable code of reproducing issue. After you fill all placeholders "{ }" fields with information and code, you can zip this file and attach it to Jira ticket of the core issue. After that, consult with your Team Lead and get his approval. Team Lead will pass this to the Core team and prioritize it along with PM.
Please, don't touch structure of this plugin since we want every plugin to look same and to keep consistency.
